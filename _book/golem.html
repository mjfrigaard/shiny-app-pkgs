golem

  TLDR

  1. If you’re comfortable developing Shiny apps1 and R packages2, golem can make your life easier.

  2. If you’re comfortable developing Shiny apps, but haven’t built many R packages, golem can streamline a lot of package development steps for you (but you’ll probably still have to go back and learn at some point).

  3. If you’re new to Shiny and comfortable developing R packages, you might find modules challenging, but there is still a lot to be ganied by adopting golem.

  4. If you’re new to Shiny and R package development, start with Mastering Shiny and the Shiny documentation.

Code 
install.packages("golem")
library(golem)

  1. If your shiny apps have modules, you’re (at least) an ‘intermediate’ developer.↩︎

  2. If you’ve written a package and successfully shared it with others, you’re (at least) an ‘intermediate’ package developer.↩︎
